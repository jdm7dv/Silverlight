Kit 3D

Kit3D is a 3D graphics engine for creating 3D content in Microsoft silverlight. The engine is currently written in Javascript. For more information about the project you can visit the CodePlex site, this contains extra documentation and also the source code, so if you feel like particiapting or if you just want to use the code for your own project then feel free: http://www.codeplex.com/Kit3D
